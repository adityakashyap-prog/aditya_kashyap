Rubrics:

1. A complete React.js single-page application is delivered with all 13 required pages/sections built and accessible via defined routes.
2. The application is deployed live on Vercel.
3. A public GitHub repository is provided with a clean commit history.
4. The live Vercel URL loads without a 404 or build error.
5. The Global Navigation bar includes all 9 links: Discover, Collections, Genres, Community Hub, Newsroom, Support, Search, Wishlist, and Profile.
6. The Global Navigation remains fixed and visible on all pages while the user scrolls.
7. The Global Navigation includes a hamburger menu that opens and closes the nav links on mobile viewports.
8. The Hero Showcase is a carousel containing between 3 and 5 featured fictional games from the approved game list.
9. Each Hero Showcase slide displays a hero artwork image, game title, genre badge, star or numeric rating, a short description of 2 to 3 lines, and an Explore button linking to that game's details page.
10. The Hero Showcase carousel autoplays without any user input required.
11. The Hero Showcase carousel supports manual navigation via Swiper.js prev and next controls.
12. The Discover Collections section contains horizontal sliders for all 6 collections: Editor's Picks, Story-Driven Adventures, Open World Experiences, Competitive Challenges, Community Favorites, and Hidden Gems.
13. All collection sliders scroll horizontally using Swiper.js.
14. All game cards inside collection sliders remain fully visible within the slider viewport without clipping or overflow.
15. The Explore Genres section displays a grid of all 10 genre cards: Action, Adventure, RPG, Racing, Strategy, Sports, Simulation, Sci-Fi, Fantasy, and Indie, each with custom visual artwork.
16. The Trending Arena section includes tabs or filters for all 4 views: Trending Games, Community Favorites, Most Viewed Titles, and Recommended Experiences.
17. Each Trending Arena game card displays a cover artwork image, rating, genre label, and platform information.
18. The Play Zone section displays exactly 3 game cards: Memory Matrix, Quiz Quest, and Cyber Runner, each with the title, genre, and description matching the brief.
19. Each Play Zone card displays a visible Coming Soon visual treatment such as a muted overlay, lock icon, or status badge.
20. Play Zone cards do not appear interactive or playable in their default state.
21. Clicking any Play Zone card opens a modal popup displaying the exact Coming Soon message: This gaming experience is currently under development. Stay tuned for future updates from GameVerse.
22. The Play Zone modal can be dismissed via a close button or by clicking the background overlay.
23. The Play Zone modal open action includes a Framer Motion animation.
24. The Play Zone modal close action includes a Framer Motion animation.
25. The New Horizons section displays newly released fictional game concepts with artwork, release information, star or numeric rating, and a summary.
26. The Future Releases section displays upcoming fictional titles with preview artwork, a release window label, genre, and platform availability.
27. A Game Details page exists at the route /game/:id and loads the correct game data for each of the 10 fictional games.
28. The Game Details page includes an overview section showing title, description, genre, release info, and platform tags.
29. The Game Details page includes a media gallery displaying exactly 4 screenshot or promotional artwork images sourced from the game's JSON entry.
30. The Game Details page includes a community insights section showing a rating summary and at least 2 featured static reviews.
31. The Game Details page includes a similar titles section showing related game cards.
32. A Wishlist Center page exists at the route /wishlist and lists all games the user has saved.
33. Clicking Add to Wishlist on any game card or game details page saves that game's id to browser local storage under the key gameverse_wishlist.
34. Removing a game from the Wishlist page removes its id from browser local storage.
35. The wishlist content renders correctly after a full browser refresh, confirming local storage persistence.
36. A Gamer Profile page exists at the route /profile displaying a user avatar, display username, favorite genre badges, wishlist item count, and a recently viewed games list.
37. A Newsroom page exists at the route /newsroom with at least 2 entries in each of the 4 sections: Industry Updates, New Announcements, Community Spotlights, and Feature Articles.
38. A Community Hub page exists at the route /community displaying popular discussions, community recommendations, trending topics, and featured games as display-only content with no user submission functionality.
39. All game data for the 10 fictional games and the 3 Play Zone entries is loaded from local JSON files and is not hardcoded directly inside React components.
40. The layout renders without horizontal overflow, clipped text, or overlapping elements at viewport widths of 320px, 768px, and 1280px.
